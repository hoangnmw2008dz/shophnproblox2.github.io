# Roblox-Shop (Static)

Các bước đưa lên GitHub Pages:
1. Tạo repo mới (ví dụ `shoproblox` hoặc `username.github.io`).
2. Upload toàn bộ thư mục `index.html`, `css/`, `js/`, `assets/`.
3. Vào Settings → Pages → Chọn branch `main` và folder `root` → Save.
4. Chờ vài phút, truy cập `https://<username>.github.io/<repo>`.

Tùy chỉnh sản phẩm: sửa `js/script.js` → mảng `PRODUCTS`.

Chú ý: website này là static (không có giỏ hàng / thanh toán). Khi người mua bấm "Mua ngay" sẽ hiện hộp modal kèm hướng dẫn liên hệ. Bạn có thể tích hợp thêm form gửi email hoặc webhook nếu cần.
